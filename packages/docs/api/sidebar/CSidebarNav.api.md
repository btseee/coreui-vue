### CSidebarNav

```jsx
import { CSidebarNav } from '@coreui/vue'
```

#### Props

| Prop name | Description                                                                             | Type                | Values | Default |
| --------- | --------------------------------------------------------------------------------------- | ------------------- | ------ | ------- |
